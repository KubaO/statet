New Integrated Console

The new integrated console is separated in two areas: the input line and the output viewer. The input line allows you to submit directly input to launched tool. The output viewer shows all previews input and "their response", the output of the tool.

Input

You can type commands in input line text field and edit code almost like in an usual editor of StatET. Pressing ENTER or the 'Submit'-Button sends your current input to your tool.

There are some special keyboard shortcuts to access the history and to scroll the output viewer, if you are in the input line:

History control
Up Goto the following older item in the history
Down Goto the following newer item in the history
Tip: You can use the Undo feature to go back to the last edited input.
Output control
Shift+Up Scroll output one line up
Shift+Down Scroll output one line down
Shift+PageUp Scroll output one page up
Shift+PageDown Scroll output one page down
Shift+Ctrl+Home Shift+Command+Home Scroll output to the start
Shift+Ctrl+End Shift+Command+End Scroll output to the end
