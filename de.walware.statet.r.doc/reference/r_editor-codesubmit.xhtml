Submit Actions

StatET allows easy interaction with an R console: chunks of code can be submitted by means of simple keyboard shortcuts. The following paragraphs detail some individual submit actions that are part of the StatET R editor.

Print variable value to console

When the cursor is on the name of a given variable, it is possible to print the value of the variable to the console. The default keyboard shortcuts for this command are Ctrl+R Ctrl+P Ctrl-R P

Submit current line or selection

Ctrl+R Ctrl+R Ctrl+R R

Submit entire command

Ctrl+R Ctrl+E Ctrl+R E

Submit function

When the cursor is inside a function definition, it is possible to submit the entire function using Ctrl+R Ctrl+F Ctrl-R F This comes of course very handy when one operates small changes to a function definition and wants to quickly test the use of the newly changed function.

Submit and paste results

One can submit a block of R statements and directly paste the R output into the script file using Ctrl+R Ctrl+V Ctrl+R V

Source in current file

When editing an R script, it is possible to source in the entire file using Ctrl+R Ctrl+S Ctrl+R S
