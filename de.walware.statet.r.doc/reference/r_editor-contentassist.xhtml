Content Assist

Use of Content Assist

Content Assist is a mechanism within Eclipse that offers to insert predefined code snippets from a menu that is activated on a simple keyboard shortcut, Ctrl+Space by default. After pressing the shortcut, the following menu appears:

As can be seen above, the snippet to insert can be chosen from the window. Right from the items in the menu a popup displays the code that will be inserted for the particular snippet that is selected from the menu. Pressing Enter will insert the snippet in the editor and allow you to further edit the snippet.

Configuration of snippet templates

The content of the predefined snippets can be configured in Window > Preferences at StatET > Source Editors > R Templates
