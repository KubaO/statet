R Console (Page/View)

- New Integrated Console -

The R console (new integrated console) page is separated in two areas: the input line and the output viewer. The input line allows you to submit directly input to launched tool. The output viewer shows all previews input and "their response", the output of R.

Cmd Input

You can type commands in input line text field and edit code almost like in an usual editor of StatET. Pressing ENTER or the 'Submit'-Button sends your current input to your tool.

Special shortcuts

There are some special actions and keyboard shortcuts to access the history and to scroll the output viewer, if you are in the input line:

History control
Go to the following older history item Up 1
Go to the following newer history item Down 1
Go to the newest history item (last input) Ctrl+Alt+End 2
Search for the following older history item starting with the text before the cursor. Ctrl+Alt+Up 2
Search for the following newer history item starting with the text before the cursor. Ctrl+Alt+Down 2
Tip: You can use the Undo feature to go back to the last edited input.
Output control
Scroll output one line up Shift+Up 1
Scroll output one line down Shift+Down 1
Scroll output one page up Shift+PageUp 1
Scroll output one page down. Shift+PageDown 1
Scroll output to the start Shift+Ctrl+Home Shift+Command+Home 1
Scroll output to the end Shift+Ctrl+End Shift+Command+End 1

1 Keyboard binding is fix.
2 Keyboard binding can be changed as usual.

Content Assistant

The command line offers the possibility to select file paths using a graphical interface. This functionality is available whenever the cursor is positioned within quotes by using Ctrl+Space.

Console Output

  • Coloring of errors
  • Links to line in script when submitting from script

R Console Toolbar

As any other view, the R Console View offers a toolbar with buttons and menus that allow straightforward interaction and configuration.

Cancel Tasks 

Cancel Tasks is a menu that allows to cancel the execution of the currently running R task. This can be useful e.g. when one launches an intensive simulation, but discovers the parameters were not correctly specified. The menu offers three options.

Terminate R

The Stop Console button allows to quit a given R. On pressing this button, the user will be prompted to save the workspace or not.

Clear Console 

The Clear Console button will clean the output window of the console. After clearing the console, it obviously is no longer possible to scroll up to previous output.

Scroll-Lock 

The Scroll-Lock button is a toggle button that allows to lock scrolling of the console output window. This implies that when submitting new statements either at the command line or from a script, the view the user has on the output window will not change. It is, of course, always possible to manually scroll to a given location within the output window.

Pin Console 

If one pins a console by toggling the Pin Console button, it means that other consoles that might be opened (R Consoles or SVN consoles, TeXlipse consoles, etc.) will not open inside the current Console View, but inside a separate, new Console View. The pinning of an R console can also be set when configuring an R Launch Configuration.

Console Button

If several consoles are opened within one Console View, the console button allows to navigate to other consoles that are available within the Console View. When two consoles are open, a single click allows to navigate between the two consoles.

Open New Console 

R Console Menu 

The R Console menu is available in the right corner of the R Console toolbar. It currently offers the user two items, which are treated in a separate paragraph below.

Change Working Directory

This menu item allows to change the R working directory of the console via an Eclipse interface.

Adjust Output Width

This menu item will change the number of characters of an R output line, such that it perfectly fits the current dimensions of the R Console View.
