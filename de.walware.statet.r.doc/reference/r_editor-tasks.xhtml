Tasks

Tasks are Eclipse's way of keeping track of small notes that are part of the code. StatET defines two different task keywords that will allow Eclipse to highlight the tasks in the code and offer the information in a Tasks View:

  • TODO
  • FIXME

The meaning of the Task tags is self-explanatory. The tasks will be recognized as such if they are preceded with a single comment character and a single space, as in # TODO: remove hard-coding of colours

Tasks are a type of Eclipse Annotations. Therefore, an individual task icon will be displayed in the left gutter. In the right gutter, which gives an overview of all Eclipse annotations, the task annotations are indicated by small blue rectangles. Clicking on such a rectangle will position the cursor on the corresponding task.

One can edit and add or remove task keywords in the preferences at StatET > Task Tags
